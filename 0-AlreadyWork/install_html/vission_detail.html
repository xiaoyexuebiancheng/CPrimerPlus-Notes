1.初学者安装什么编译器比较好呢？

答：如果先前有入门书的，建议新手安装书上推荐的编译器。如果先前没有入门书的话小叶建议安装VS2017（从入门到有能力工作至少两年时间，企业通用的编译器一般都是前三到前七年的稳定版编译器）。 不过没关系，小叶会对所列的这四种编译器的基本使用方法进行讲解

2.安装不成功怎么办？

答：现在通用的C语言入门书都是以VC6为模板编译器进行讲解的，而VC6在老早在win7还没普及就不更新了，更没有了官网，现在网上得到的VC6也在安装上经常出现各种兼容问题。所以小叶为你们准备了四个，总能有安装成功的，排除一切阻力学编程(ง •_•)ง

（ps:当年小叶我开始学的时候为了下载能用的VC6，找了整整一个星期才找到，下了不行就删，再找再删，想想我当时为了学编程也是蛮有毅力的(●ˇ∀ˇ●)）

3.编译器要安装特别全面的功能吗？

答：没有必要，安装的足够学习足够就好了，安装很多很多插件不用也只是占用着硬盘空间，特别是VS2017/2019全部工具安装完要占200多G，下载就够呛(っ °Д °;)っ。所以只要安装够用够学习就好啦。
